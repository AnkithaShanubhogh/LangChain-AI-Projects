# LangChain-AI-Projects
A collection of AI Chatbot and RAG projects using LangChain and Groq
  # 🤖 LangChain AI Project Suite

  This repository showcases three levels of AI implementation using **LangChain**,    **Groq**, and **OpenRouter**.
  
  ## 📁 Projects
  * **Chatbot.ipynb**: A basic conversational AI.
  * **RAG.ipynb**: Document-based AI that answers questions from PDFs.
  * **RAG-with-History.ipynb**: Advanced RAG system that maintains conversation       context.
  
  ## 🛠️ Setup
  1. Clone this repo.
  2. Install dependencies: `pip install langchain-openai langchain-groq python-dotenv`.
  3. Create a `.env` file with your `GROQ_API_KEY`.
